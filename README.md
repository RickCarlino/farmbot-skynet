farmbot-skynet
===========

This is a Ruby library that interface the Farmbot driver into http://skynet.im

Features
--------

 * It's got websockets and stuff.

Requirements
------------

* Raspberry pi running Ruby.
* A complete Farmbot assembly

Install
-------

* bundle install

Run it!
-------

 * `ruby farmbot.rb`
